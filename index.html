Machine Learning Project ↗

The primary objective was to develop an optimized supervised learning algorithm capable of accurately recognizing and classifying handwritten digits. Utilizing the "load digits" dataset from the sci-kit learn library. The team performed extensive data engineering and preprocessing, including image partitioning, Sobel filters, and Principal Component Analysis (PCA) to reduce dimensionality while preserving key information. By implementing automated pipelines and comparing different models, the project aimed to achieve high classification precision through linear separation principles such as One-versus-One (OvO) and One-versus-Rest (OvR).

Linear Solver ↗

This project focuses on solving high-dimensional linear systems (Ax = b) by implementing key iterative methods, including Jacobi, Gauss-Seidel, and SOR (both dense and sparse versions). Originally developed in Python, the system was re-engineered in C++ using Object-Oriented Programming (OOP). By leveraging inheritance and polymorphism, the architecture was optimized for code reusability and efficiency. This transition allowed for a performance benchmark, comparing Python's flexibility against C++'s native memory management and execution speed
